# Data & AI Job Market Analytics

This project analyzes global salaries for Data Science, AI, and Machine Learning roles using Python, PostgreSQL, SQL, and Tableau.

## Tech Stack
Python  
PostgreSQL  
SQL  
Tableau  

## Analysis Performed
- Data cleaning using Python (Pandas)
- Storage and querying with PostgreSQL
- Salary trend analysis using SQL
- Data visualization using Tableau

## Dashboard Insights
- Executive-level roles earn the highest salaries (~$226k average)
- Machine Learning and Data Science roles dominate the top-paying jobs
- Senior-level salaries are significantly higher than entry and mid-level roles
- AI-related roles show strong compensation growth across experience levels
